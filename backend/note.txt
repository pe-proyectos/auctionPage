1. User
Create User → Registrar nuevo usuario. ✅

Read Users → Listar todos o buscar por ID/email. ✅

Update User → Editar datos personales o cantidad de fichas.✅

Delete User → Eliminar un usuario.✅

Extra:

Recargar fichas.

Consultar saldo de fichas.

2. Categoria
Create Categoria → Crear nueva categoría (ej. Autos, Motos).

Read Categorias → Listar todas o buscar por ID/nombre.

Update Categoria → Editar nombre.

Delete Categoria → Eliminar categoría (opcional, pero ojo con relaciones).

3. Publicacion
Create Publicacion → Crear nueva subasta con datos (título, precio, fecha, usuario, categoría).

Read Publicaciones → Listar todas o filtrar por categoría/usuario.

Update Publicacion → Editar datos (solo si no ha iniciado la subasta o bajo reglas).

Delete Publicacion → Eliminar una subasta.

4. Puja
Create Puja → Registrar una puja (requiere descontar fichas y validar saldo).

Read Pujas → Listar pujas por usuario o por publicación.

Update Puja → No suele actualizarse, pero podrías permitir corregir errores antes del cierre.

Delete Puja → Cancelar puja (opcional y con reglas estrictas).
